# FreeRTOS 示例

此示例工程演示了FreeRTOS任务创建及调度，需要使用FreeRTOS的可以参考此示例。

## 示例说明

此示例基于 ```TB-02_Kit``` 开发板创建了三个任务，两个LED闪烁，一个串口输出。

运行后RGB灯快速闪烁，冷暖灯慢速闪烁。串口每秒输出一次```Hello FreeRTOS！```。

按下开发板上的```K_D2```,RGB灯停止闪烁，再次按下后RGB灯回复闪烁。

## 配置说明

必须在工程目录下的```makefile```文件中配置如下信息：

    #如果使用RTOS，必须将这个设置为 1
    USE_FREE_RTOS = 1
